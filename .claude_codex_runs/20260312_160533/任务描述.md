# 任务描述

## 目标
创建 "Renee 的职场小妙招" — 一个双重项目：
1. **Claude Code Skill**：检测 LLM 偷懒/摆烂行为，通过职场 PUA 话术"激励"其完成任务。包含 contract 系统、多 agent 检查、评分、语义匹配偷懒短语。
2. **reneepang.com 网站**：展示这个 skill 的落地页，部署到 Cloudflare Pages。

## Deliverables Contract（交付物合同）

| ID | 交付物 | 验收标准 |
|----|--------|---------|
| D1 | 网站首页 (`src/App.jsx`) — 展示 Renee 职场小妙招 skill 的完整落地页，包含品牌设计、功能介绍、PUA 话术展示 | `npx vite build` 成功且 `dist/index.html` 存在 |
| D2 | PUA 话术数据库 (`plugins/renee-pua/phrases.json`) — 包含 6 类 PUA 话术（否定、比较、画饼、内疚、威胁、ownership），每类至少 8 条中英双语 | 文件存在且为有效 JSON，包含至少 48 条话术 |
| D3 | 偷懒模式检测数据 (`plugins/renee-pua/lazy-patterns.json`) — LLM 常见偷懒/摆烂短语的语义匹配规则库，中英双语 | 文件存在且为有效 JSON，包含至少 40 条匹配模式 |
| D4 | Skill 命令文件 (`skill/renee.md`) — 完整的 Claude Code skill 命令，包含 contract 创建、子项目拆分、agent 派遣、评分、偷懒检测、PUA 回复逻辑 | 文件存在且包含 `## Contract 创建` 和 `## 偷懒检测` 等核心章节 |
| D5 | 网站样式组件 (`src/components/`) — 至少包含 Hero、Features、PhraseShowcase、ContractDemo 四个组件 | `src/components/` 下至少有 4 个 `.jsx` 文件 |
| D6 | Cloudflare Pages 部署配置 (`wrangler.toml`) | 文件存在且包含 `pages_build_output_dir = "dist"` |

## 角色与分工
- Coordinator (Claude Code): plans, dispatches, QA verification, contract enforcement
- Executors (codex-1, codex-2): execute tasks, self-check
- QA (Claude Code): independent verification of all deliverables after coding

## 技术栈
- Stack: javascript-frontend (JavaScript Frontend)
- 工具链: eslint, prettier, vitest, vite

## 运行约束
- Max parallel workers: 2
- Max batch size: 2
- Max orchestration rounds: 20
